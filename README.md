<h1>AJAX, FETCH with Node.js Workshop.</h1>

- M01 — Reverse a string by sending it to the server and displaying the reversed result. (Technologies: XHR, Node.js)
- M02 — Send two numbers to the server and display their sum. (Technologies: XHR, Node.js)
- M03 — Perform arithmetic operations on inputs via the server and show results. (Technologies: XHR, Node.js)
- M04 — Search for a number on the server and return whether it is found. (Technologies: XHR, Node.js)
- M05 — Retrieve an image from the server and display it using XHR. (Technologies: XHR, binary response, Node.js)
- M06 — Fetch an image as a Blob and display it using an object URL. (Technologies: XHR/Blob, URL.createObjectURL, Node.js)
- M07 — Submit a form via XHR to the server and display the server response. (Technologies: XHR2, FormData, Node.js)
- M08 — Upload form data with progress feedback during the XHR transfer. (Technologies: XHR, FormData, XHR upload progress events, Node.js)
- M10 — Demonstrate creating and using a basic JavaScript Promise. (Technologies: Promises)
- M11 — Show promise error handling using .catch(). (Technologies: Promises, error handling)
- M12 — Chain multiple promises to perform sequential asynchronous steps. (Technologies: Promises chaining)
- M13 — Run multiple promises in parallel and handle results (Promise.all). (Technologies: Promise.all)
- M14 — Use Promise.race to obtain the fastest promise result. (Technologies: Promise.race)
- M15 — Demonstrate async functions and await for asynchronous code. (Technologies: async/await)
- M16 — Chain async/await calls to perform sequential asynchronous operations. (Technologies: async/await)
- M20 — Perform a fetch request to retrieve data from the server. (Technologies: Fetch API, Node.js)
- M21 — Use fetch with async/await to get and process server responses. (Technologies: Fetch API, async/await, Node.js)
- M22 — Demonstrate Server-Sent Events (SSE): server pushes messages and client receives them. (Technologies: SSE, EventSource, Node.js)
